# 🎵 Нейро-Саундтрек

**Нейро-Саундтрек** — это веб-приложение, которое помогает пользователю подобрать музыку на основе текстового описания настроения или события. Сервис анализирует запрос с помощью нейросети GigaChat и возвращает подборку релевантных треков и плейлистов со ссылками на Яндекс Музыку.

## 🚀 Основной функционал (MVP)

* **Интеллектуальный анализ**: Обработка естественного языка (например, «песня для пробежки утром») для поиска музыки.
* **Тематическая фильтрация**: Система жестко ограничена музыкальной тематикой и отклоняет нецелевые запросы (например, вопросы о программировании или бытовых советах).
* **Интерактивный интерфейс**: Главный экран с текстовым полем, кнопкой отправки и индикацией процесса обработки запроса.
* **Результаты поиска**: Выдача карточек, содержащих названия треков и активные ссылки на Яндекс Музыку.

## 📦 Артефакты проекта

* **Репозиторий на GitHub**: Полный исходный код приложения (Frontend + Backend).
* **Демонстрационная версия**: Работающее веб-приложение, доступное для локального запуска.

## 🛠 Технические требования

### Frontend
* **Главная страница**: Поле ввода с плейсхолдером «Опишите настроение или событие».
* **Экран результатов**: Отображение списка из 3 плейлистов на одной странице.
* **Обработка состояний**: Визуальный индикатор загрузки и сообщения об отсутствии результатов, если подходящие треки не найдены.

### Backend
* **Интеграция с GigaChat API**: Серверная часть обеспечивает отправку запросов и получение ответов от нейросети.
* **Логика валидации**: Настройка модели на игнорирование запросов, не связанных с музыкой.

## ✅ Критерии готовности

* Приложение корректно обрабатывает целевые запросы (например, «музыка для того, чтобы погонять по трассе»).
* При вводе нецелевого текста (например, «как почистить картошку») приложение выводит сообщение: «По вашему запросу ничего не найдено».
* В случае, когда запрос целевой, появляется список из трёх треков и двух плейлистов, есть возможность перейти на следующую страницу, где будут представленны другие треки и плейлисты (тематика не меняется).
* К каждому треки и плейлесту прикреплена уникальная ссылка, которая открывает новую вкладку с соответсвующей страницей в Яндекс Музыки.
